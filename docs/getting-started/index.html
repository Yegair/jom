Installation

jom is distributed as a Maven package via the central repository. The most recent version is 0.0.1-alpha.5.

Gradle#

build.gradle.kts
repositories {
mavenCentral()
}
dependencies {
implementation("io.yegair.jom:jom:$jomVersion")
}

Maven#

pom.xml
<dependency>
<groupId>io.yegair.jom</groupId>
<artifactId>jom</artifactId>
<version>${jom.version}</version>
</dependency>
